# 다중 상속
하나의 클래스가 여러개의 클래스를 상속받는 것.

ex)
```cpp
class MusicPhone : public MP3, public MobliePhone {
~~~
}
```
여기서 MusicPhone클래스는 MP3클래스 / MobliePhone클래스를 다중 상속받는다.

MuiscPhone클래스는 기본 클래스들의 멤버들을 모두 호출할 수 있다.(기본 클래스의 멤버가 public일때만 ㅎ)

---
# 가상 상속
### 다중 상속의 문제점
다중 상속을 하다보면, 만약 어떤 멤버함수가 특정한 클래스2개에 중복되어있다고 가정해보자.

이 특정한 2개 클래스를 상속받은 파생클래스에서 이 멤버함수를 호출하면, 어떤 클래스에 있는 **함수를** 

**호출하고자 하는 건지 모호하다.**

## 그래서 가상 상속이 필요하다
가상 상속을 한다면, 

파생 클래스가 객체를 생성할 때, 

기본 클래스의 멤버공간을 한번 할당하고 -> 그 공간을 함께 공유한다.
